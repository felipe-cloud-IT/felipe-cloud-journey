# Networking Basics

## What I learned today:

- What is an IP address
- What is DNS
- What is DHCP
- Difference between LAN and WAN

## Key concepts:

- IP = device identifier
- DNS = name to IP translation
- DHCP = automatic IP assignment

## Real example:
When I connect to internet, DHCP assigns my IP automatically.
DNS allows me to open websites like google.com.
